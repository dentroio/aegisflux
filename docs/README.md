## 🔄 Documentation Maintenance

This documentation is maintained alongside the codebase. When making changes:

1. **Update relevant documentation** for any API or behavior changes
2. **Add new sections** for new features or capabilities
3. **Archive old documentation** instead of deleting it
4. **Keep examples current** and test them regularly

## 📝 Contributing to Documentation

1. **Follow the existing structure** and formatting
